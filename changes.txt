------------ Версия 1.1 ------------
- добавлен KeyInputCallback для ввода с клавиатуры и соответствующий код для этого.
- удалено определение uint и uchar, чтобы не тащить ничего лишнего, и потенциально не вызывать никаких конфликтов с другими подобными определениями.
- добавлен класс WindowController, теперь логика управления окном перенесена на классы.
- добавлена функция GetCurrentPlatform() для идентификации платформы, на которой запущен код.
- Добавлена функция SetNativeWindowPosition() для установки положения окна.
- Добавлена функция CloseNativeWindow() для принудительного закрытия окна.
- Добавлено множество сообщений об ошибках, выводящихся в консоль, для упрощения отладки.
- переименовано:
	struct GLContextCreateInfo -> GLContextInfo
- Добавлена функция GetGLCoreVersion() для получения текущей версии библиотеки.
