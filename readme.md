# CoreGL - библиотека для управления окнами и инициализации OpenGL
данная библиотека дает платформонезависимый механизм для управления окнами и инициализации OpenGL.
<br>
Помимо проекта библиотеки репозиторий тоже содержит CoreGLTest, который содержит пример кода и используется для тестирования.
<br>
Фичи:
- Полноценная работа с окнами системы: создание, перемещение, обработка событий и др.
- Загрузка OpenGL контекста и OpenGL функций.
- С-style интерфейс, за счет которого можно подключать библиотеку куда-угодно.
<br>
<br>

Текущие проблемы:
- Реализован только бекенд для Windows и проект привязан к Visual Studio.
- Загрузка OpenGL функций привязана к контексту в библиотеке, необходимо это разделить, чтобы библиотеку можно было использовать с другими библиотеками, управляющими окнами, например GLFW.
- Работа с окнами не перенесена на классы (возможно, стоит создать WindowController или что-то типа того).
- Не решен (и непонятно, как решать) вопрос с локализацией названий окна, отличимых от английского. Пока что в Windows бекенде, в StringToWstring захардкожена кодировка, поддерживающая русский язык.
- несмотря на то, что есть возможность выбирать стандарт OpenGL, нет никакого отсечения по версии при загрузке функций. Поэтому все отвественность ложится на пользователя.
- Присутствуют файлы glext.h и wglext.h, они не нужны проекту, но используются для взятия сигнатур функций и типов.
<br>

Смотри исходники для получения документации.
